# D08 R3.3 — Classification explicite des références

Les références D08 ne deviennent pas canon parce qu'elles plaisent. Le owner peut désormais les
classer explicitement (`poster_energy`, `world_style`, `filter_reference`, `output_template`,
`anti_pattern`, `rejected`) sans déclencher de génération ni changer leur source.

Le classement reste privé, traçable et sans effet provider, asset, export ou canonisation.
