# Living Truth Spine — Canon to Git delivery

Le contrat canon est découpé sans promesse fictive :

1. source/version/change ledger ;
2. class/cohort/student/alias registry ;
3. correction context profile et preuves versionnées ;
4. asset/reference registry D08 ;
5. release/backup/incident/recovery registry.
