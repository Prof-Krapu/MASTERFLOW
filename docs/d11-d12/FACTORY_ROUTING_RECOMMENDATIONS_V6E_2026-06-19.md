# D11 — Factory Routing Recommendations V6E

V6E expose une recommandation de routage en lecture seule pour chaque candidate update.

- `DA` suggère D08 ;
- `PROJECT_LORE` suggère D09 ;
- `PEDAGOGY` propose D05/D06 à arbitrer ;
- les autres classifications n'obtiennent aucune recommandation automatique.

La recommandation ne renseigne jamais `target_domain` et ne déclenche ni routage, canon,
Usage Harvester, action, job, import ou activation runtime.
